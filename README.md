# playwrightSuiteLoop
This project was part of an application for a position at Loop Software and Testing services to gauge my playwright knowledge amongst other things. The project was written using javascript and Playwright and is data-driven by creating dynamic and scalable code drawing from a single JSON file.

## Description
The project navigates to a Kanban style website and validates that certain tasks are present in the correct columns with the correct tag(s).

## Installation
Only the playwright npm package is necessary i.e. npm install playwright
To run, use npx playwright test 
    
## Usage
The page object files are used to feed functions into the main test file with data being handled by a singular JSON file.
